# Treatiseresearch

＊＊＊＊＊コミットの命名規則＊＊＊＊＊＊

ファイル名：oooo_yyyymmdd:説明

ooooの中身は
*********************************************
・feat　追加機能

・fix　 エラー修正

・ref   リファクタリング

・docs　コードに関係ない文書等
**********************************************
の命名規則でコミットする事

********************注意*********************
同じ機能のコードや
同じ文書はまとめて一つのファイルにまとめる事
